# Security

## Reporting a vulnerability

Report suspected vulnerabilities via GitHub's private [security advisories](https://github.com/po4yka/industrial-design-cmp/security/advisories/new) for this repository. Do not open a public issue.

Include: affected version, reproduction steps, impact, and — if known — a suggested fix.

Expect an acknowledgement within 7 days. Fixes ship in a patch release; severe issues may ship out-of-band.

## Supported versions

Pre-1.0 — only the latest minor receives fixes. A supported-versions matrix will be added at 1.0.
