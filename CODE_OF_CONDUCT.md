# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Summary

Participate in a way that makes the project welcoming to all contributors — regardless of experience level, background, or identity.

Harassment, discriminatory language, personal attacks, and unconstructive hostility are not tolerated. Disagreement on technical or design decisions is expected and encouraged; keep critique focused on ideas and artifacts, not on people.

## Reporting

Report violations via a private GitHub security advisory on [po4yka/industrial-design-cmp](https://github.com/po4yka/industrial-design-cmp/security/advisories) or by DM to the repository owner.

## Enforcement

Maintainers may edit, remove, or reject contributions (issues, comments, commits, code, wiki edits, pull requests) that do not align with this Code of Conduct, and may temporarily or permanently ban contributors for repeated or severe violations.

## Scope

This Code of Conduct applies within all project spaces — issues, pull requests, discussions, releases — and when representing the project in public spaces.
